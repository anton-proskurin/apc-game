# APC - Журнал разработки

## Проект

**APC** — браузерная игра-симулятор взрослой жизни для 8-летнего Марка. Подарок от папы.

## Технические решения

### Выбор стека
- Vanilla JS (ES6 modules) без фреймворков — простота, нулевое время сборки
- Единый CSS файл с CSS-переменными — лёгкость поддержки
- localStorage для сохранения — работает оффлайн
- Emoji вместо изображений — мгновенная загрузка, веселье для ребёнка

### Архитектура
- Hash-based SPA роутер — переходы без перезагрузки
- Центральное состояние с event emitter — реактивные обновления UI
- Паттерн сцен (render/cleanup) — чистый жизненный цикл
- Авто-сохранение каждые 30 секунд + при переходах

### Мобильная оптимизация
- Max-width 480px с центрированием
- Touch-friendly: все кнопки >= 44x44px
- Safe area insets для iPhone X+
- Отключён zoom по двойному тапу

## Локальная разработка

```bash
# Запуск сервера
python3 -m http.server 8080

# Открыть в браузере
open http://localhost:8080
```

## Деплой

```bash
# Деплой на Yandex Cloud Object Storage
./deploy.sh
```

Требуется настроенный `yc` CLI и бакет в Object Storage.
